# Kubernetes Learning Path

A comprehensive guide covering Kubernetes fundamentals, AWS EKS integration, and production deployment strategies.

## What You'll Learn

| Chapter | Topic | Description |
|---------|-------|-------------|
| 01 | [What is Kubernetes](./01-what-is-kubernetes) | Core concepts, architecture, and how Kubernetes orchestrates containers |
| 02 | [EKS Setup on AWS](./02-eks-setup) | Step-by-step guide to provisioning and configuring Amazon EKS |
| 03 | [Deployment Strategies](./03-deployment-strategies) | Rolling updates, canary deployments, and progressive delivery |

## Prerequisites

- Basic understanding of **containers** and **Docker**
- An **AWS account** with appropriate permissions
- Familiarity with **YAML** syntax
- **kubectl**, **AWS CLI**, and **eksctl** installed locally

## How to Use This Guide

1. Start with **Chapter 01** to build a solid foundation of Kubernetes concepts
2. Move to **Chapter 02** to set up a real EKS cluster on AWS
3. Finish with **Chapter 03** to learn how to safely deploy applications using rolling and canary strategies
