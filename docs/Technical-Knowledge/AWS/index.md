# ☁️ AWS Learning Path

A structured roadmap to achieve the **AWS Solutions Architect Associate (SAA-C03)** certification and build deep cloud engineering skills.

**Start here →** [SAA-C03 Study Guide](./SAA_C03/index.md) — comprehensive exam preparation covering all four domains, every in-scope service, and exam strategies.

---

## 🎯 Certification Target: SAA-C03

| Detail | Info |
|--------|------|
| **Exam** | AWS Certified Solutions Architect – Associate (SAA-C03) |
| **Duration** | 130 minutes, 65 questions (50 scored + 15 unscored) |
| **Passing score** | 720 / 1,000 |
| **Cost** | $150 USD |
| **Validity** | 3 years |

### Exam Domains

| Domain | Weight | Focus Areas |
|--------|--------|-------------|
| 1. Design Secure Architectures | **30%** | IAM, encryption, VPC security, Organizations, compliance, federation |
| 2. Design Resilient Architectures | **26%** | Multi-AZ, Auto Scaling, DR strategies, decoupling, containers, serverless |
| 3. Design High-Performing Architectures | **24%** | Compute, storage, databases, caching, CDN, data analytics, streaming |
| 4. Design Cost-Optimized Architectures | **20%** | Pricing models, right-sizing, storage tiers, network costs, cost tools |
