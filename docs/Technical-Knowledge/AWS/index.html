☁️ AWS Learning Path

A structured roadmap to achieve the AWS Solutions Architect Associate (SAA-C03) certification and build deep cloud engineering skills.

Start here → SAA-C03 Study Guide — comprehensive exam preparation covering all four domains, every in-scope service, and exam strategies.

🎯 Certification Target: SAA-C03​

DetailInfo
ExamAWS Certified Solutions Architect – Associate (SAA-C03)
Duration130 minutes, 65 questions (50 scored + 15 unscored)
Passing score720 / 1,000
Cost$150 USD
Validity3 years

Exam Domains​

DomainWeightFocus Areas
1. Design Secure Architectures30%IAM, encryption, VPC security, Organizations, compliance, federation
2. Design Resilient Architectures26%Multi-AZ, Auto Scaling, DR strategies, decoupling, containers, serverless
3. Design High-Performing Architectures24%Compute, storage, databases, caching, CDN, data analytics, streaming
4. Design Cost-Optimized Architectures20%Pricing models, right-sizing, storage tiers, network costs, cost tools

📍 Study Roadmap​

WeekFocusWhat to Do
1–2Core ServicesRead Global Infrastructure, IAM, Compute, Storage, Networking, Databases
3–4Integration & SecurityRead App Integration, Security Services, Monitoring, Analytics, Migration
5Architecture PatternsRead HA & DR, Cost Optimization, Exam Strategy
6–7Practice ExamsTake 4–6 timed practice exams. Identify and review weak areas.
8Final Review & BookRe-read weak sections. Book exam when scoring 80%+ consistently.

📖 Study Guide Chapters​

Foundation​

  • 01 — Global Infrastructure — Regions, AZs, Edge Locations, Local Zones, Wavelength, Outposts
  • 02 — IAM & Secure Access — Users, roles, policies, SCPs, Organizations, SSO, STS, federation

Core Services​

  • 03 — Compute — EC2, Auto Scaling, ELB, Lambda, ECS/EKS, Fargate, Beanstalk, Batch
  • 04 — Storage — S3, EBS, EFS, FSx, Storage Gateway, Snow Family, Backup, DataSync
  • 05 — Networking — VPC, SG, NACL, Route 53, CloudFront, Global Accelerator, Direct Connect, VPN
  • 06 — Databases — RDS, Aurora, DynamoDB, ElastiCache, Redshift, specialty databases, DMS

Integration & Security​

  • 07 — Application Integration — SQS, SNS, EventBridge, Step Functions, API Gateway, AppSync, MQ
  • 08 — Security Services — KMS, WAF, Shield, GuardDuty, Macie, Cognito, Secrets Manager, ACM
  • 09 — Monitoring & Governance — CloudWatch, CloudTrail, Config, SSM, CloudFormation, Trusted Advisor

Specialized Topics​

  • 10 — Data Analytics — Kinesis, Athena, Glue, EMR, Redshift, QuickSight, OpenSearch, Lake Formation
  • 11 — Migration & Transfer — DMS, Snow Family, DataSync, Application Migration, Transfer Family, ML services

Architecture Patterns & Exam Prep​

  • 12 — High Availability & DR — Multi-AZ/Region patterns, DR strategies, RPO/RTO, immutable infrastructure
  • 13 — Cost Optimization — Pricing models, right-sizing, storage tiers, network costs, cost tools
  • 14 — Exam Strategy — Study plan, question techniques, common traps, key numbers
