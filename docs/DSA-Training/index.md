# DSA Intensive Training Roadmap

A **20-week, 420-problem** intensive program designed for intermediate-to-advanced engineers targeting Big Tech coding interviews. Each week is a self-contained module with theory, code templates, curated problems, and mock interviews.

---

## Program Overview

| | |
|---|---|
| **Duration** | 20 weeks (5 months) |
| **Pace** | ~3 problems/day, 21 problems/week |
| **Total Problems** | 420+ (Medium to Hard) |
| **Mock Interviews** | 2 per week (40 total) |
| **Prerequisites** | Comfortable with arrays, basic trees, basic DP, sliding window, two pointers |

## How to Use This Roadmap

1. **Follow the interactive tracker** at [DSA Roadmap](/dsa-roadmap) to check off problems and monitor progress
2. **Read the theory section first** in each weekly page — even if you think you know the topic
3. **Do problems in order**: Foundation (Day 1-2) → Intermediate (Day 3-4) → Advanced (Day 5-7)
4. **Complete the mock interview** at the end of each week under timed conditions (20 min per question)
5. **Review follow-up questions** — interviewers frequently extend problems with these

:::tip Daily Schedule Suggestion
- **Morning (1h)**: Read theory + study worked example
- **Afternoon (1.5h)**: Solve 2 problems with full implementation
- **Evening (45min)**: Solve 1 harder problem or review solutions
- **Weekend**: Mock interview practice + revisit unsolved problems
:::

## Progress Milestones

| Milestone | When | What You Should Be Able To Do |
|-----------|------|-------------------------------|
| Checkpoint 1 | End of Week 4 | Recognize monotonic patterns, parametric BS, DSU problems instantly |
| Checkpoint 2 | End of Week 8 | Solve any graph/tree problem up to Hard; comfortable with segment trees |
| Checkpoint 3 | End of Week 13 | Handle all DP variants including bitmask, digit, tree, and optimized DP |
| Checkpoint 4 | End of Week 19 | String algorithms, number theory, combinatorics, game theory — no blind spots |
| Final | End of Week 20 | Consistently solve 2 Medium-Hard problems in 45 minutes under pressure |
