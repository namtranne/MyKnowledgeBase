# DSA Foundations — 20-Week Training Roadmap

A structured, beginner-to-intermediate curriculum designed to build strong fundamentals for coding interviews. Starts with core data structures, progresses through trees, graphs, and algorithms, then introduces moderate advanced topics. Each week includes full theory lessons, code templates, 21 curated problems (Easy/Medium/Hard mix), and 2+ mock interviews.

---

## Program Overview

| Detail | Value |
|--------|-------|
| **Duration** | 20 weeks (140 days) |
| **Pace** | ~3 problems/day, 21/week |
| **Total Problems** | 420 |
| **Mock Interviews** | 2+ per week (42 total) |
| **Difficulty Mix** | ~25% Easy, ~50% Medium, ~25% Hard |
| **Prerequisites** | Basic programming in any language (Python recommended) |
| **Target Audience** | Beginners preparing for coding interviews at any tech company |

## How to Use This Roadmap

1. **Read the theory** — Each week starts with a full lesson assuming zero prior knowledge of the topic.
2. **Study the templates** — Code templates give you reusable patterns for each technique.
3. **Solve 3 problems/day** — Problems are grouped by difficulty tier within each week.
4. **Do the mock interviews** — Simulate real interview conditions with follow-up questions.
5. **Track your progress** — Use the [interactive tracker](/dsa-roadmap) to check off problems.

:::tip Daily Schedule Suggestion
- **Morning (1 hr):** Read theory + study 1 worked example
- **Afternoon (1.5 hrs):** Solve 2–3 problems
- **Evening (30 min):** Review solutions, note patterns
:::
