## 🚀 Getting Started

1. **Audit yourself** — scan each section's checklist and mark topics you're weak at
2. **Prioritize** — Networking + Databases + System Design + OS are highest-impact for FAANG
3. **Follow the learning paths** — each section has a recommended reading order
4. **Practice actively** — use the interview questions chapter in each section
5. **Prepare stories** — 6–8 behavioral stories mapped to leadership principles
6. **Use the search bar** (top-right) to find specific topics quickly

:::tip Suggested 8-Week Study Plan
- **Weeks 1–2:** Networking, OS, Concurrency
- **Weeks 3–4:** Databases, Storage Systems, Transactions
- **Weeks 5–6:** System Design, Distributed Systems, Caching
- **Weeks 7–8:** Security, Reliability, Observability, Mock Interviews + Behavioral Prep
:::
