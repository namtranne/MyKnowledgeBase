# Repository Summary

## Overview

Personal knowledge base documentation site. Markdown notes on DSA, system design, AWS, Kafka, OS, and technical book summaries. Includes interactive pages (DSA roadmap tracker, algorithm visualizations).

## Tech Stack

- Docusaurus 3.8.1 (React static site generator)
- React 18.3.1 + JSX
- CSS Modules + custom CSS
- Webpack (via Docusaurus)
- pnpm 10.33.0

## Entry Points

| Path | URL | Purpose |
|------|-----|---------|
| `src/pages/index.js` | `/` | Homepage (particle canvas background) |
| `src/pages/dsa-roadmap/index.js` | `/dsa-roadmap` | Interactive DSA roadmap with progress tracking |
| `docs/**/*.md` | `/docs/*` | Static documentation, sidebar auto-generated |

## Scripts

- `npm run dev` — hot-reload dev server at localhost:3000
- `npm run build` — static site to `build/`
- `npm run serve` — serve built site
- `npm run deploy` — build and push to gh-pages branch
- `npm run clear` — clear Docusaurus cache

## Core Dependencies

`@docusaurus/core` 3.8.1, `@docusaurus/preset-classic` 3.8.1, `@docusaurus/theme-mermaid`, `@easyops-cn/docusaurus-search-local`, `prism-react-renderer`, `remark-math`, `rehype-katex`, `react` 18.3.1, `react-dom` 18.3.1

## Notable Conventions

- Sidebar auto-generated from `docs/` folder structure via `sidebars.js`
- Each component is a folder with `index.js` + optional `styles.module.css`
- Dark mode is the default; toggle is disabled
- Math rendering via KaTeX (remark/rehype plugins)
- Prism supports java, python, bash, json, yaml, properties
- DSA roadmap persists progress in `localStorage`
